# ChatGPT to Markdown — Chrome拡張機能

ChatGPTのチャット内容をMarkdown（`.md`）ファイルとして保存するChrome拡張機能です。

## 対応コンテンツ

- 通常のチャットメッセージ（ユーザー・ChatGPT双方）
- コードブロック（言語タグ付きフェンスコード）
- リスト・表・リンク・数式（KaTeX）
- キャンバスドキュメント（ChatGPTのProseMirrorエディタで作成されたドキュメント）

## インストール方法

Chromeウェブストアには公開していないため、デベロッパーモードで手動インストールします。

1. このリポジトリをダウンロードまたはクローンする

2. Chromeのアドレスバーに `chrome://extensions` と入力して開く

3. 右上の **「デベロッパーモード」** トグルをONにする

4. **「パッケージ化されていない拡張機能を読み込む」** をクリックする

5. この拡張機能のフォルダ（`manifest.json` が含まれるフォルダ）を選択する

6. 拡張機能一覧に「ChatGPT to Markdown」が追加されれば完了

## 使い方

1. [chatgpt.com](https://chatgpt.com) を開き、保存したいチャットを表示する

2. ツールバーの拡張機能アイコンをクリックしてポップアップを開く
   > アイコンが見えない場合はツールバー右端のパズルピースアイコン（🧩）をクリックし、「ChatGPT to Markdown」をピン留めしてください

3. **「Markdownとして保存」** ボタンをクリックする

4. `{チャットタイトル}_{日付}.md` という名前でファイルがダウンロードされる

### ポップアップの表示

| 表示 | 意味 |
|---|---|
| 緑色のメッセージ | 保存成功（タイトルとメッセージ数を表示） |
| 赤色のメッセージ | エラー（詳細を確認してください） |

## 出力形式

```markdown
# チャットタイトル

*Exported on 2026-03-20 from ChatGPT*

---

## You

ユーザーのメッセージ内容

---

## ChatGPT

ChatGPTの応答内容

---
```

キャンバスドキュメント（ChatGPTが作成した文書）はMarkdownコードブロックとして出力されます:

````markdown
## ChatGPT

```markdown
# ドキュメントのタイトル

ドキュメントの内容...
```
````

## 注意事項

- **chatgpt.com のみ対応**しています。他のサイトでは動作しません
- ChatGPTが応答を生成中（ストリーミング中）はエクスポートできません。応答が完了してからご利用ください
- ダウンロード先はChromeのデフォルトダウンロードフォルダです

## 拡張機能を更新する場合

ファイルを変更した後、以下の操作が必要です:

| 変更したファイル | 必要な操作 |
|---|---|
| `content.js` | `chrome://extensions` で拡張機能の ↻ をクリック（タブのリロード不要） |
| `popup.html` / `popup.js` | 拡張機能をリロード → ポップアップを閉じて再度開く |
| `manifest.json` | 拡張機能をリロード |
